Conseils d'utilisation

Sont accessibles ici des e-books aux formats EPUB, AZW3 et PDF, qui ne se valent pas tous.
Le format EPUB est un standard supporté par la quasi-totalité des liseuses et lecteurs d'e-books. Sur iOS, l'app préinstallée iBooks peut les lire. Sous Android, si il n'y a aucun lecteur préinstallé, je peux vous conseiller ReadEra, sur le Play Store, qui permet de lire un nombre considérable de formats d'e-books ainsi que les PDF. Presque toutes les liseuses supportent ce format nativement (pas les Kindle...). Ce format est à préférer si vous le pouvez, car la conversion est ici globalement de meilleure qualité, et vous pouvez adapter la taille de police et d'autres paramètres qui améliorent le confort de lecture.

Le format AZW3 est totalement équivalent au EPUB si ce n'est que c'est le format d'Amazon, lisible sur les applications et les liseuses Kindle.

Si vous ne pouvez pas utiliser un de ces deux formats, il reste les PDF. Ceux-ci sont moins pratiques à lire sur mobile ou liseuse car vous devrez généralement zoomer dessus pour voir quelque chose. De plus, la conversion est moins bonne, ce qui peut créer notamment quelques problèmes de mise en page, qui ne dépendent pas de moi. Cependant, le format PDF est lisible sur à peu près n'importe quel appareil doté d'un écran.
